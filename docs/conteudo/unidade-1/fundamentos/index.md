# Fundamentos

Em programação competitiva, problemas recebem dados de entrada e exigem dados de saída. Isso significa:

**Entrada**:
dados fornecidos ao programa.

**Saída**:
resposta produzida pelo programa.

Nesta seção, ainda não veremos processamento de dados. Por enquanto, focaremos em como o Python costuma pedir e mostrar dados.

## Objetivo

Ao final desta seção, você deverá ser capaz de:

- usar input()
- usar print()
- capturar múltiplos inputs
