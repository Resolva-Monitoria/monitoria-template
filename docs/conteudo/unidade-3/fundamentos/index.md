# Fundamentos

Em programação competitiva, muitas tarefas exigem repetir um bloco de código várias vezes. Isso significa:

**Laço[text](controle_loop.md)**:
estrutura que repete a execução de um trecho de código.

**Repetição**:
controle de quantas vezes ou até que condição algo deve ser executado.

Nesta seção, ainda não veremos problemas complexos. Por enquanto, focaremos em como o Python costuma criar e controlar loops.

## Objetivo:

Ao final desta seção, você deverá ser capaz de:

- usar for com range()
- usar while com condições
