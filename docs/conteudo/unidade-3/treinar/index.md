# Praticar

A melhor forma de aprender laços de repetição é resolvendo problemas reais.  
Aqui você encontra atividades da OBI para treinar `for` e `while`.

**Nível fácil**: problemas simples para fixar o conceito.  
**Nível difícil**: problemas que exigem mais raciocínio e lógica.

## Objetivo

Ao final desta seção, você deverá ser capaz de:

- resolver problemas usando `for`
- resolver problemas usando `while`
- escolher qual laço usar conforme o problema

---

## Como Usar Este Material

1. **Comece pelos fáceis**: resolva pelo menos 2 problemas do nível fácil
2. **Entenda a lógica**: não copie código, entenda o raciocínio
3. **Avance gradualmente**: só passe para o difícil quando sentir confiança

---

## Dicas

- Use `for` quando souber **quantas vezes** repetir
- Use `while` quando não souber o número de repetições antecipadamente
- Sempre verifique a **condição de parada**
- Teste com os **exemplos** da questão antes de submeter
