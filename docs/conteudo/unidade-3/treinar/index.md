# Praticar

A melhor forma de aprender vetores é resolvendo problemas reais. Aqui você encontra atividades da OBI para treinar manipulação de vetores.

---

## Como usar este material

1. **Escolha o nível**: Comece pelos fáceis
2. **Resolva o problema**: Tente fazer sozinho
3. **Compare com a solução**: Veja onde errou
4. **Avance gradualmente**: Só passe para o difícil após dominar os fáceis
