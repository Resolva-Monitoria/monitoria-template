# Praticar com `for`

Laços `for` são ideais quando **sabemos quantas vezes** precisamos repetir algo.

## Quando Usar `for`

| Situação | Exemplo |
|----------|---------|
| Percorrer intervalo | testar todos os números de 1 a 100 |
| Processar dígitos | somar dígitos de um número |
| Contar ocorrências | contar quantos números têm soma X |

## Dicas

- Use `range(inicio, fim + 1)` para incluir o valor final
- Para cada elemento, processe **toda** a lógica antes de passar para o próximo
- Lembre-se de **resetar** contadores dentro do loop
