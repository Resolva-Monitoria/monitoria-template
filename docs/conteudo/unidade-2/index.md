# Unidade 2 — Condicionais

Nesta unidade, você aprenderá os fundamentos de estruturas de decisão em Python.

Em síntese, estruturas de decisão são diferentes comandos que serão utilizados variando entre diferentes resultados. A essas estruturas damos o nome de `condicionais`.

---

## Objetivos

Ao final, você deverá ser capaz de:

- Utilizar `if/else` para definir condicionais
- Utilizar `match/case` como alternativa
