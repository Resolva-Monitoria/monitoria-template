# Fundamentos

Em programação competitiva, algumas respostas podem variar conforme a entrada. Isso significa que precisaremos de estruturas dentro do código que vão definir o que será retornado a depender do que vai entrar, seguindo as regras definidas no problema. Estas estruturas são chamadas de condicionais.
No Python (assim como em váias outras linguagens), a estrutura de decisão padrão que será utilizada é o `if/else`, que consiste no seguinte:

**IF**:
Condição para executar determinado código.

**ELSE**:
O que será feito caso a condição não seja atendida.

Ainda existe outra estrutura chamada `match/case`, que segue um padrão parecido. Ambas as estruturas serão apresentadas nesta unidade.

## Objetivo

Ao final desta seção, você deverá ser capaz de:

- usar `if/else`
- usar `match/case`
- mostrar resultados que variam conforme a entrada
