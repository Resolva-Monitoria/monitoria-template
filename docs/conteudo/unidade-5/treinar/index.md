# Praticar

A melhor forma de aprender dicionários é resolvendo problemas reais. Aqui você encontra atividades da OBI para treinar manipulação de dicionários.

---

## Como usar este material

1. **Escolha o nível**: Comece pelos fáceis
2. **Resolva o problema**: Tente fazer sozinho
3. **Compare com a solução**: Veja onde errou
4. **Avance gradualmente**: Só passe para o difícil após dominar os fáceis

## Dicas

- Use dicionários para **contar** elementos
- Use `dict.get(chave, padrão)` para inicialização segura
- Use `max(dict, key=dict.get)` para encontrar o mais frequente
- Itere com `.items()` quando precisar de chave e valor
