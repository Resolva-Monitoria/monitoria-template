# Fundamentos

Dicionários são estruturas de dados que armazenam pares **chave-valor**. Enquanto listas usam índices numéricos, dicionários permitem usar valoresarbitrários como chaves, tornando a busca muito eficiente.

---

## Por que aprender dicionários?

Em programação competitiva e desenvolvimento de software, muitas tarefas exigem:

- Contar frequências de elementos
- Mapear valores a chaves únicas
- Criar tabelas de pesquisa rápida
- Agrupar dados por категоis

---

## Objetivo

Ao final desta seção, você deverá ser capaz de:

- Criar dicionários em Python
- Acessar e modificar elementos
- Iterar sobre chaves, valores e itens
- Resolver problemas de contagem
