# Visão geral do ambiente de programação

> *"Antes de programar, é importante entender o ambiente."*

---

## Sobre esta página

Esta seção apresenta uma **visão geral** das ferramentas que você vai utilizar para programar em Python. O objetivo é oferecer um panorama claro e integrado dos componentes essenciais do ambiente de desenvolvimento, mostrando como eles se conectam e colaboram para transformar seu código em programas funcionais.

Aqui você ainda não vai instalar nada — apenas entender **como tudo se conecta** antes de partir para a configuração prática do ambiente.

---

## O que você vai usar

Para começar a programar em Python, você precisa de três componentes principais que trabalham em conjunto:

### Python

A linguagem de programação propriamente dita.

- **Responsável por executar seu código** – interpreta as instruções escritas e as converte em ações executáveis pelo computador
- **Interpreta os arquivos `.py`** – todo arquivo com extensão `.py` é reconhecido e processado pelo interpretador Python
- **Já vem com várias funcionalidades prontas** – bibliotecas padrão que oferecem recursos para manipulação de texto, operações matemáticas, acesso a arquivos, entre outros

O Python atua como o "motor" que dá vida aos seus programas, sendo indispensável para qualquer atividade de desenvolvimento na linguagem.

---

### Editor de código (VS Code)

É onde você escreve, organiza e gerencia seus programas.

- **Facilita a leitura e organização do código** – com destaque de sintaxe, numeração de linhas e estrutura visual clara
- **Oferece sugestões automáticas** – autocompletar de comandos, fechamento automático de parênteses e colchetes, dicas de parâmetros
- **Permite executar o código facilmente** – botões integrados e atalhos de teclado para rodar programas sem sair do editor
- **Extensões específicas para Python** – depuração, testes, formatação automática e gerenciamento de ambientes virtuais

O VS Code (Visual Studio Code) é um dos editores mais utilizados no mercado por sua leveza, potência e vasto ecossistema de extensões.

---

### Terminal

É onde você executa comandos diretamente no sistema operacional.

- **Rodar programas Python** – comando `python nome_do_arquivo.py` para executar seus scripts
- **Instalar bibliotecas** – utilizando o gerenciador de pacotes `pip` para adicionar funcionalidades extras ao Python
- **Navegar entre arquivos** – comandos como `cd` (change directory) e `dir` (Windows) ou `ls` (Linux/Mac) para se locomover pelas pastas
- **Verificar versões instaladas** – `python --version` e `pip --version` para confirmar configurações

O terminal pode parecer intimidador no início, mas alguns comandos básicos são suficientes para a grande maioria das tarefas iniciais em programação.

---

## Como tudo funciona junto

O fluxo básico é este:

```
Você escreve código → VS Code
        ↓
Salva arquivo (.py)
        ↓
Executa no terminal
        ↓
Python roda o código
        ↓
Resultado aparece na tela
```

---

## Exemplo simples

Arquivo:

```python
print("Olá, mundo!")
```

Execução no terminal:

```bash
python teste.py
```

Saída:

```
Olá, mundo!
```
