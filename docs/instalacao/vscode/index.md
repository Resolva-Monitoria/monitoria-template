# Instalando e configurando o VS Code

> *"Um bom editor não faz o programador, mas ajuda muito."*

---

## Sobre esta página

Aqui você aprenderá a instalar e configurar o **Visual Studio Code (VS Code)** para programar em Python.

---

## Instalação

!!! info "Download"
    Acesse o site oficial:

```
https://code.visualstudio.com/
```

Baixe e instale normalmente.

---

## Extensão Python

!!! tip "Extensão essencial"
    Após instalar o VS Code:

1. Abra o VS Code  
2. Pressione `Ctrl + Shift + X`  
3. Pesquise por:

```
Python
```

4. Instale a extensão da Microsoft  

---

## Configurando o interpretador

!!! info "Selecionar Python"
    Pressione:

```
Ctrl + Shift + P
```

Digite:

```
Python: Select Interpreter
```

Escolha a versão instalada.

---

## Executando código

!!! success "Rodando programa"
    Crie um arquivo:

```
teste.py
```

Código:

```python
print("Olá, VS Code!")
```

Execute:

```
Botão ▶
ou
python teste.py
```

---

## Terminal integrado

!!! tip "Terminal"
    Abra com:

```
Ctrl + J
```

---

## Conclusão

!!! success "Ambiente completo"
    Agora você possui:

- VS Code instalado  
- Python integrado  
- Ambiente profissional configurado
