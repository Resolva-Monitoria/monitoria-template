# Padrões de Codificação OBI — Python

Nesta unidade, você aprenderá boas práticas de programação específicas para resolver problemas da OBI utilizando Python.

---

## Sobre esta unidade

Aqui apresentamos **padrões recomendados** que ajudam a:

- Evitar erros comuns em competições
- Escrever código mais claro e eficiente
- Adaptar soluções ao estilo exigido pela OBI

Esses padrões não são obrigatórios em todos os casos, mas segui-los aumenta suas chances de sucesso durante a prova.

---

## O que você verá

- Formato de entrada e saída
- Organização do código
- Uso adequado de variáveis e estruturas de controle
