## Sobre o Projeto

A **Monitoria de Informática do IFC Araquari** apoia estudantes no desenvolvimento de habilidades fundamentais em programação, com foco na **Olimpíada Brasileira de Informática (OBI)**.

Este repositório reúne materiais didáticos, exercícios, conteúdos teóricos e práticos utilizados nas aulas. O objetivo é oferecer um espaço completo para estudo e evolução contínua.

---

## O que é a OBI?

A **Olimpíada Brasileira de Informática (OBI)** é uma competição nacional que estimula o raciocínio lógico e o pensamento computacional.

A competição aborda:

| Tema                  | Descrição                                  |
|-----------------------|--------------------------------------------|
| Lógica de Programação | Construção de algoritmos e sequências      |
| Resolução de Problemas| Interpretação e modelagem de situações     |
| Estruturas de Dados   | Organização e manipulação de informações   |
| Algoritmos            | Estratégias eficientes de solução          |
